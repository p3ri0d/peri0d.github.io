发表于 2019-06-25 | 分类于 渗透 ， 信息搜集

信息搜集的流程

  • 域名：whois, 子域名, 备案信息
  • 服务器 : dns 信息, 端口信息, 真实 IP
  • web : 网站架构, 敏感目录及敏感信息, 源码泄露, 旁站查询, C 段查询, 指纹系统, 探测 waf
  • 企业 : 天眼, 维基

whois 查询

  • 直接进行 whois 查询
  • 到微步查询询历史whois，可以获取到历史的whois信息
  • 反查whois，获取该注册者/电话/邮箱下的相关域名
